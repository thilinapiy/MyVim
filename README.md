MyVim
=====

Dump of my .vimrc and .vim directory

Get a clone of the repository
    
    git clone https://github.com/thilinapiy/MyVim.git

Change the directory to MyVim

    cd MyVim

Move vimrc to .vimrc

    mv vimrc .vimrc
Move vim to .vim
    
    mv vim .vim
Put both to your home directory
    
    mv .vim .vimrc ~/
